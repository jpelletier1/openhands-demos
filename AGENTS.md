# Repository Notes

- This repo contains prototype demos; prefer JavaScript/HTML/CSS and use Node.js if backend code is needed.
- The Angular 1.x shopping demo lives in `example_refactoring/public/`.
- The Angular 20 refactor demo lives in `example_refactoring/angular20-app/`.
- To verify the Angular 20 refactor, run `npm install` and `npm run build` inside `example_refactoring/angular20-app/`.
- `REFACTORING_SUMMARY.md` should stay aligned with the actual Angular 20 implementation details.
